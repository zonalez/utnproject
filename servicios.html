Servicios

Contamos con un equipo de profesionales altamente capacitado para desarrollar tareas técnicas y administración de proyectos para distintos sectores de IT.

Help Desk: Contamos con una mesa de ayuda disponible las 24 horas para atención de reclamos telefónicos y soluciones de primer nivel. Si el problema requiere mayor análisis se otorgará un número de ticket y tiempo estimado de solución

Soporte técnico remoto: El primer nivel de soporte puede resolver incidentes de forma remota durante las 24 horas.

Soporte técnico on-site: Nuestro equipo de soporte tecnico especializado esta disponible las 24 Hs. para resolver los incidentes que no hayan podido ser solucionados por nuestro equipo de help Desk. El mismo se encargara de contactarse con el cliente para solicitar mas datos si asi fuese necesario para resolver mas rapidamente su problema

Gestión de incidentes: Contamos con un software de gestión de incidentes basado en ITIL que garantiza una efectiva atencion entre nuestros equipos, permitiendo que la solucion y comunicacion de problema sea mas efectiva
